Enterprise Network Configuration Lab

I configured a full enterprise network in Cisco Packet Tracer based on a three-tier hierarchical design across two office sites. This included VLANs, Layer 2/3 EtherChannel, HSRP for gateway redundancy, RSTP with BPDU Guard and PortFast, OSPF routing with ASBR redistribution, and core network services including DHCP, DNS, NTP, SNMP, Syslog, and NAT. I also applied ACLs to secure traffic between VLANs. GITHUB

Skills: OSPF, VLANs, HSRP, RSTP, ACLs, DHCP/DNS, NAT, Cisco Packet Tracer

Small Office Network Segmentation

I built a small office network in Cisco Packet Tracer, segmenting three departments (Admin/IT, Finance/HR, CS/Reception) into separate VLANs with subnetted IP addressing. I configured a router for inter-VLAN routing, deployed wireless access points per department, and verified VLAN isolation and connectivity across the network. GITHUB

Skills: VLANs, Subnetting, Inter-VLAN Routing (ROAS), Wireless Access Points, Cisco Packet Tracer, Static IP Addressing
